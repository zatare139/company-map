# Public Companies Near Me

**🌍 Live map: https://zatare139.github.io/company-map/**

Interactive map of U.S. public companies (NYSE, Nasdaq, OTC) plotted by HQ
location. Filter by radius, exchange, market cap, and industry sector. Mark
companies you've "looked into" for cold-call prospecting.

Open `index.html` (a static site — no build step). Data is baked into
`data/companies.js`.
